# This is the smart gripper project, based on Raspberry Pi PICO 2.


## Code
There are C files.
You need to build it and then upload uf2 file into Pico2
https://github.com/VGlukhov-git/pico2/tree/main/gripper/src



## CAD
There is one STEP assembly file with all details.
https://github.com/VGlukhov-git/pico2/tree/main/gripper/cad

## USE

The idea is to use UART communication between any PC - Windows, Mac, Raspberry Pi and etc.
For communication between Windows and Pico, Mac and Pico - you can start using by connection Micro USB

Sample file to communicate by Micro-USB - test.py
